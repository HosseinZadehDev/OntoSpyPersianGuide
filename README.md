# راهنمای فارسی Ontospy

یک صفحهٔ آموزشی فارسی و واکنش‌گرا برای معرفی، نصب و استفاده از ابزار خط فرمان **Ontospy** در ویندوز. این پروژه شامل آموزش متنی، ویدئوی آموزشی، نمونه‌دستورها و کتابچهٔ راهنمای قابل دانلود است.

## امکانات

- آموزش نصب Python 3.11 و Ontospy
- معرفی دستورهای اصلی مانند `scan`، `gendocs`، `lib`، `shell`، `ser` و `utils`
- ویدئوپلیر سفارشی با کنترل پخش، صدا، زمان، سرعت و تمام‌صفحه
- طراحی راست‌چین و واکنش‌گرا برای دسکتاپ، تبلت و موبایل
- امکان کپی سریع دستورها
- کتابچهٔ راهنمای PDF قابل دانلود
- معرفی منابع رسمی و نمونه‌های قابل استفاده

## ساختار پروژه

```text
.
├── documents/              # کتابچهٔ راهنمای PDF
├── fonts/                  # محل نگهداری فونت‌های محلی
├── icons/                  # آیکون‌های رابط کاربری
├── images/                 # تصاویر و پوستر ویدئو
├── scripts/
│   └── index.js            # منطق تعاملی صفحه و ویدئوپلیر
├── styles/
│   └── index.css           # استایل‌ها و قواعد واکنش‌گرا
├── videos/                 # فایل‌های ویدئویی آموزشی
├── index.html              # صفحهٔ اصلی
└── README.md
```

## اجرای پروژه

پروژه به build یا نصب وابستگی نیاز ندارد. فایل `index.html` را مستقیماً در مرورگر باز کنید.

برای اجرا از طریق یک وب‌سرور محلی نیز می‌توانید در ریشهٔ پروژه دستور زیر را اجرا کنید:

```powershell
python -m http.server 8000
```

سپس آدرس زیر را باز کنید:

```text
http://localhost:8000
```

## دریافت مخزن

ویدئوی اصلی پروژه با **Git LFS** مدیریت می‌شود. پیش از clone مطمئن شوید Git LFS نصب و فعال است:

```powershell
git lfs install
git clone <repository-url>
```

اگر مخزن را قبلاً دریافت کرده‌اید اما ویدئو دانلود نشده است، اجرا کنید:

```powershell
git lfs pull
```
